# Committing Basics Exercise

1. Create a new folder called `CampingTrip`
2. Initialize a new Git repository inside the `CampingTrip` folder (make sure you're not already inside a Git repository!)
3. Create a new file called `gear.txt`
4. Create another new file called `food.txt`
5. Make a commit that includes both empty files. The commit message should be:

   ```text
   create gear and food lists
   ```

6. In the `gear.txt` file, add the following:

   ```text
   - 1 tent
   - 2 sleeping bags
   ```

7. In the `food.txt` file, add the following:

   ```text
   - 8 granola bars
   - 4 apples
   - 1 jar peanut butter
   ```

8. Make a new commit, including **ONLY the changes from the `food.txt` file.** The commit message should be:

   ```text
   add snacks for camping trip
   ```

9. Make a second commit including **ONLY the changes to the `gear.txt` file.** The commit message should be:

   ```text
   add essential camping gear
   ```

10. Add the following line to the end of `food.txt`:

    ```text
    - 1 bag marshmallows
    ```

11. In the `gear.txt` file, change the first line so that it says **2 tents** instead of **1 tent**:

    ```text
    - 2 tents
    - 2 sleeping bags
    ```

12. **Make a commit that includes the changes to BOTH files.** The commit message should read:

    ```text
    add supplies for campfire night
    ```

13. **Use a Git command to display a list of commits. You should see something similar to:**

    ```text
    add supplies for campfire night
    add essential camping gear
    add snacks for camping trip
    create gear and food lists
    ```
